# Trabalho Prático de Aplicação Web e Móveis

Este projeto é uma aplicação web responsiva desenvolvida como parte da minha graduação em Engenharia de Software. 
O objetivo principal foi criar uma interface moderna e funcional que servisse como portfólio pessoal e centralizador 
de projetos.

## Funcionalidades

- Interface responsiva adaptável a diferentes tamanhos de tela.
- Sistema de temas (Claro e escuro).
- Player de áudio integrado com feedback visual.
- Navegação estruturada por seções com efeito de scroll suave.
- Divisores visuais e padronização de espaçamentos.

## Tecnologias Utilizadas

- **HTML5**: Estruturação semântica.
- **CSS3**: Estilização com variáveis CSS, Flexbox e Grid.
- **JavaScript**: Manipulação da DOM, mudança de temas e lógica de controle de áudio.
- **Google Fonts**: Integração das fontes Roboto e Caveat para identidade visual.

## O que eu aprendi

Este projeto foi fundamental para consolidar meus conhecimentos em desenvolvimento front-end, permitindo-me:

- Entender a importância da Arquitetura de Informação ao separar seções por cores e paddings.
- Gerenciar estados de interface (como o alternador de temas) via JavaScript puro.
- Desenvolver um olhar crítico para UX, substituindo links automáticos por interações mais controladas.
- Estruturar layouts complexos utilizando apenas CSS nativo, sem dependência de frameworks externos.
